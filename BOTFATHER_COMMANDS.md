# Bot Commands Setup for BotFather
# Professional English descriptions for all bot commands

## Setting Commands in BotFather

Open @BotFather in Telegram and follow these steps:

### 1. Select your bot
Send: `/mybots` → Select your bot

### 2. Set Commands
Send: `/setcommands`

### 3. Paste the following command list:

```
start - Launch bot and access main menu
help - View available commands and help
income - Record new income transaction
expense - Record new expense transaction
balance - Check current balance
stats - View financial statistics
transfer - Send money to another user
transfers - View your sent transfers
received - View transfers you received
settings - Manage app preferences
currency - Change default currency
language - Switch interface language
report - Generate financial report
cancel - Cancel current operation
menu - Return to main menu
```

## Detailed Command Descriptions

### Core Commands:
- **start** - Launch bot and access main menu

BotFather also supports language-specific command descriptions:

**For Russian:**
